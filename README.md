Bootstrap Layout Collection

This repository contains a collection of commonly used website layouts built with Bootstrap. It is designed for students and beginners who want to learn how Bootstrap structures pages and how various layouts are implemented using its grid system, utility classes, and responsive features.

The purpose of this project is to provide clear, practical examples that can be studied, reused, or extended.

Contents

This project includes the following layout types:

Fixed Layout

Fluid Layout

Responsive Layout

Adaptive Layout

Grid-Based Layout

Single-Page Layout

Card Layout

Card-Based Section Layout

Split-Screen Layout

Multi-Column Layouts

Stacked Column Layouts

Additional variations demonstrating Bootstrap container and grid usage

Each layout is created using Bootstrap 5 and structured for clarity so learners can easily understand how the classes work.

Technologies Used

HTML5

CSS3

Bootstrap 5.3

Responsive design principles

How to Use

Clone or download the repository

Open any of the HTML files in your browser

Study the structure of each layout

Modify or extend the layouts as needed

Use the code as reference when building your own responsive pages

Purpose of This Project

This project is intended for:

Students learning Bootstrap

Beginners practicing responsive web design

Individuals looking for ready-made layout templates

Anyone wanting to understand Bootstrap grid and layout behavior through examples
